# Fake QQ UI

全方位模仿 QQ NT 的聊天气泡样式与功能

> [!NOTE]  
> 全方位模仿：大部分样式来自与 QQ NT for Windows，部分样式经修改

## 已实现

- 纯文本消息
- 图文消息
- 图片消息
- 图片文件消息
- 文件消息
- 引用（回复）消息
- 语音
  - 点击播放
  - 进度条显示音频音量大小
  - 显示音频时长
  - 动态音频气泡长度
  - 播放动画
  - 播放时点击暂停
  - 右键显示“语音转文本”内容

## 注意事项

- 本项目大量 HTML 和 CSS 来自 QQ
- 内置的 **QHeader** 与 **QMain** 组件不一定适合所有人，你可以使用自己的聊天窗口样式包裹消息组件
- 气泡颜色默认为白色，因此背景建议使用很淡的灰色
